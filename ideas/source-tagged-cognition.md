# Source-Tagged Cognition

## Premise

The brain runs without a reliable provenance system. Whether a representation arose from sensory input, recall, imagination, inference, or external suggestion is decided post hoc by a separate fallible process. Confabulation, false memory, suggestion susceptibility, déjà vu, source-attribution failures in psychiatric conditions, and the felt indistinguishability of vivid memory and vivid imagination all sit on this missing layer. An interface that exposes and writes source tags onto cognitive representations turns cognition from untyped content into typed content. The amplification is not faster thinking, larger working memory, or new substrate. It is metadata. Every thought arrives with a verifiable provenance, and downstream processes that consume the thought can condition on its source.

## Mechanism

The substrate for source distinction already exists in primate cortex, in an underdeveloped state. Layer-specific activity differs between bottom-up perception and top-down imagery in early sensory regions, measurable with high-density recording. Hippocampal-cortical replay during recall carries sharp-wave-ripple signatures absent during initial encoding. Top-down prediction descends through layer 5/6 pyramidal projections; bottom-up evidence ascends through layer 4. Corollary-discharge circuits tag self-generated motor commands so the resulting sensory consequences are attenuated, and the failure of this circuit is the most defensible neural correlate of auditory hallucination. Prefrontal source-monitoring, particularly in anterior PFC, weights these signals into a felt sense of source — slowly, late, and unreliably under load.

The interface reads layer-resolved activity in the cortical columns hosting a representation, classifies the source as perception, recall, imagery, inference, transmitted, or externally imposed, and writes the tag by gating the projection from the representation's region into the prefrontal source-monitoring circuit and the parietal working-memory hub. Tag writing is not modification of the underlying representation. It is modification of the metadata channel that downstream consumers already sample, with the interface's classifier substituting for the endogenous classifier's slow, error-prone verdict. The user does not see a label flash. They experience the representation with a different felt-source quality, the way vivid memory and vivid imagination are already felt-different, but now veridical.

Two regimes. Passive tagging continuously classifies and projects tags without user intervention; representations enter working memory pre-tagged. Active query lets the user point at a current representation and request re-evaluation under a stricter, slower classifier, for cases where the passive verdict is uncertain. The tag space is extensible: as new cognitive sources become reachable (subagent-3, REM-bout-of-2026-04-12, transmitted-from-A), the classifier acquires classes, and downstream consumers learn to condition on them. The defensible minimum schema is binary (internal vs external); the interesting schema is dozens.

The bidirectional asymmetry matters. Reading source signatures is an inverse problem: the same column firing pattern may have different sources, and the classifier resolves this from timing, layer profile, oscillatory context, and broader network state. Writing tags is easier than writing representations because the target is smaller and downstream of computation. But the write must arrive in working memory together with the representation, not after. A tag that lags the representation by tens of milliseconds is forensic; one that lags by single milliseconds is protective.

## Composition

Source-tagged cognition is a safety wrapper that several other directions need but do not contain. Direction 3 (random-access memory) becomes auditable: a recalled episode is tagged retrieval, reconstruction, or fabrication, and the random-access store ceases to be a vector for high-fidelity false memory. Direction 4 (concept-level IO) addresses its loss-of-interpretive-layer risk: transmitted patterns arrive tagged foreign, and the receiver decides whether to integrate. Direction 5 (parallel subagents) loses its identity-fragmentation failure mode: thoughts surfacing into the main thread carry subagent-of-origin tags. Direction 7 (extra cortex) gains a trust gradient: output from grown tissue is tagged as such until the user chooses to promote it. Direction 8 (editable motivation) gains its missing audit primitive: installed preferences are tagged installed. Prior surgery gains an answer to its differential-auditability problem: installed priors carry installation tags persistently. Dream-compute closes the imported-confabulation channel: REM-generated content cannot integrate into memory as if perceived, because the tag persists through consolidation. Critical-period re-entry gains its missing defense against hyper-suggestibility: content acquired during an open window is tagged acquired-while-open, and the user retains the ability to discount it under closed conditions. Source-tagging does not improve direction 2 (subjective time), which is upstream of source attribution, and contributes little to direction 1 (new senses), where the source is already implicit in the modality. The through-line: inferred provenance → addressable provenance.

## What would falsify it

- If source signatures are not consistently decodable from activity above the noise floor at the spatial and temporal scales reachable by any plausible interface — perception and vivid imagery use the same effective neural code at every level a probe can sample — then classification is guesswork and tags are confabulations of their own.
- If the brain's downstream consumers of representations cannot make use of an external tag — the felt-source quality is generated entirely by the endogenous source-monitoring circuit, and projecting a tag into it does not override its own verdict — then tags exist as a forensic readout but do not change cognition.
- If accurate classification latency exceeds the latency at which representations influence decisions, the tool is forensic rather than protective, and the central use case (defending against suggestion in real time) is unavailable.
- If imposing tags causes the endogenous source-monitoring circuit to degrade through disuse, the way habitual GPS use degrades cognitive mapping, then net source-attribution resilience drops over time and users without the interface running are worse off than users who never had one.
- If the schema-correct distinctions are not the ones the brain implements — the brain distinguishes representations along axes the user cannot consciously use (familiarity, arousal, valence), and the perception/recall/imagery taxonomy is a folk-psychological category with no neural reality at the level a classifier can read — then no meaningful tag schema exists and the project fails at ontology.
- If tag-writes are themselves writable adversarially with no asymmetric defense — the same channel that marks a real perception verified will mark a fabricated representation verified under attacker control — then tags are a more potent attack surface than the suggestion they were meant to defeat, and the technology is net negative.
- If source tags do not survive memory consolidation — a representation tagged imagined at encoding is tagged remembered after a night of sleep, because consolidation strips metadata — then the most valuable long-horizon use cases collapse and the tool degrades to a real-time alarm.

## Open questions

- How many discrete source classes can the brain reliably emit and consume? Two (internal/external) is a binary alarm; six (perception/recall/imagery/inference/transmitted/imposed) is the interesting regime; dozens may exceed what downstream consumers can act on.
- Should tags decay with the representation, or persist independently in a separate memory store? Persistent tags allow long-term audit but require their own indexing primitive; decaying tags match the brain's natural metadata behavior but reintroduce the failure modes the tool exists to solve.
- What is the user-facing affordance for active tag query? An overlay rendered in concept-IO format, a modulation of the felt-source quality, a peripheral signal, or no surfaced representation at all.
- Do tags propagate through inference? If A is tagged imagined and B is derived from A, does B inherit the tag, and is inheritance computed at derivation time or at consumption time. Lazy inheritance is cheaper; eager inheritance is more reliable under reasoning chains.
- How does tagging interact with pooled cognition (direction 6)? The pool needs a tag system that survives pooling without giving one participant the ability to mark another's contributions as untrustworthy by writing tags directly.
- Does the brain require a native tag class for uncertain source, or does the interface need to force a classification in every case? Forcing classification in genuinely uncertain cases is itself a fabrication; permitting an uncertain tag may be where the user's natural skepticism survives.
- Is there a tag class for productive imagination — speculatively generated, not yet committed, deliberately not bound to source — that preserves the creative felt-quality without flagging the content as low-confidence?

## Risks

- Tag forgery as the dominant attack vector. An adversary with write access to the tag channel can mark installed content as native perception, and the user's defenses against suggestion are weaker than before because the user now trusts the tag system. The asymmetry between attack and defense determines the net direction of the technology.
- Hypervigilance pathology. Continuous awareness of source provenance may foreground a metadata layer that ordinary cognition runs more efficiently without. Some thinking depends on letting content feel real long enough to be used.
- Loss of useful confabulation. The brain confabulates to bridge gaps in self-narrative and integrate fragmentary memory into a workable whole. Rigorous tagging surfaces gaps that the brain was right to paper over, with no replacement integration mechanism available.
- Imagination flagged as deficit. If the schema does not carry a productive class for speculatively generated content, imagination is continuously rendered as low-confidence material, and the creative felt-quality is degraded in proportion to how reliably the tagger runs.
- Forensic discoverability. Tags written into long-term memory are recoverable by anyone with read access to the tag channel. Coercive disclosure of tag history, or legal subpoena, becomes a new privacy attack surface that did not exist when source attribution lived as fallible internal inference.
- Dependence without graceful degradation. After years of running with the interface, endogenous source-monitoring is under-exercised; if the interface fails, the unmediated brain is worse at the function than it would have been without the tool, and source-attribution failures cluster around the moment of disconnection.
- Schema lock-in. The tag classes shipped in the first interface define what kinds of cognitive provenance are even thinkable. A user with an interface that has no intuited class begins to treat intuitions as illegitimate, even when they are the signal the user has most reason to trust.
- Tag-as-permission creep. Institutions that accept source-tagged content as evidence (courts, employers, regulators) generate pressure to reclassify untagged or low-confidence content as unreliable by default, and the user's ordinary unaided cognition becomes inadmissible.
